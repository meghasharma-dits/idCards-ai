How the application works

Overview

Our ID Extraction Application is designed to simplify and automate the processing of financial documents. Users can upload ID cards in various formats like PDF, images (JPEG/PNG) and the application will intelligently extract and structure key data such as Name, ID Number, DOB, etc. Once extracted, this data is mapped into a structured JSON format, which users can review directly within an intuitive dashboard interface. Additionally, users can download both the mapped JSON and the original uploaded document for their records or integration into other systems.

Upload a Document

  • Click on the “Upload” button.
  • Select a file in PDF, image (JPG/PNG).
  • The application will process and extract key Id Card data.

View Extracted Data

  • Once uploaded, navigate to the Dashboard.
  • Review the extracted and structured data mapped in a clean JSON view.
  • Cross-check fields such as Name, ID Number, DOB, and more.

Download Options

  • Mapped JSON: Click “Download JSON” to get the structured Id Card data.
